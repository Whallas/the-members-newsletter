Introduction

Base URL: http://localhost:8083

Este documento tem como objetivo fornecer todas as informações necessárias para trabalhar com nossa API de Newsletters.

Conforme você rola a página, você verá exemplos de código para trabalhar com a API em diferentes linguagens de programação na área escura à direita (ou como parte do conteúdo em dispositivos móveis). Você pode alternar a linguagem usada com as abas no canto superior direito (ou no menu de navegação no canto superior esquerdo em dispositivos móveis).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

Example request:
curl --request POST \
    "http://localhost:8083/api/users/create_reader" \
    --header "Authorization: Bearer {YOUR_AUTH_KEY}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"pgclguxtrylj\",
    \"email\": \"volkman.alberta@example.com\",
    \"password\": \"jY(]Z6\\/W?9cAd15\",
    \"password_confirmation\": \"soluta\"
}"

const url = new URL(
    "http://localhost:8083/api/users/create_reader"
);

const headers = {
    "Authorization": "Bearer {YOUR_AUTH_KEY}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "pgclguxtrylj",
    "email": "volkman.alberta@example.com",
    "password": "jY(]Z6\/W?9cAd15",
    "password_confirmation": "soluta"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
